# Veno

Easy to use information gathering bot
Maintained by NixonXC

# How to Use

First replace your AUTH's value with your token in db/database.json

```json
{
    "AUTH": "your token here"
}
```

then run the main file (main.py)
And it should print:

```py
Bot is ready!
Logged in as: Bot_Name
With ID: Bot_ID
------
```
